# 🌌 Private Intelligence Framework & RAG Web Agent

A high-performance, secure data retrieval and text synthesis engine built with Streamlit and powered by cloud-hosted Groq LPUs (Language Processing Units) for blistering fast execution speed.

---

## 🚀 Key Features

- **Blistering Fast Inference:** Leverages Groq's custom hardware layer to stream open-source models (`Llama 3.3`, `Qwen 32B`) at hundreds of tokens per second.
- **Resilient Web Fallback:** Integrated with live internet search loops featuring automated retry mechanisms and exponential backoff to handle rate throttling smoothly.
- **Local Context Vector Store:** Manages persistent index collections securely, pulling right from your localized document vectors before synthesizing outputs.
- **Dynamic Model Swapping:** Interactive dropdown selector built into a high-visibility, dark-mode user interface for on-the-fly model hot-swapping.
